# ENVS-193DS_winter_2026_final
Final for envs 193DS
# General Information
This is my final project for ENVS 193DS. It goes over language used in research writing, data visualization of mean sea kelp fronds, data analysis of mountain plover site selection, and personal data analysis.
# Data and File Overview
Annual_Kelp_All_Years.csv: sourced from SBC LTER data catalogue,dataset on kelp fronds in Santa Barbara Channel.
MOPL_nest_site_selection_Duchardt_et_al_2019: dataset on mountain plover nest site selection. sourced from "Duchardt, Courtney J., JL Beck, DJ Augustine. 2020. “Mountain Plover habitat selection and nest survival in relation to weather variability and spatial attributes of black-tailed prairie dog disturbance.”""
apple.xlsx: personal dataset on apple drop rates between trees with leaves broken by player and trees with leaves left to decay.
